<h1 align=center>各地天氣</h1>

這是一個使用 Vue 開發串接第三方api的查詢各地天氣的網站，能夠搜尋全世界的城市，並且查看當地天氣

# 簡介
- 可在網站上搜尋全世界的城市，並查看現在天氣
- 查看當天每小時的天氣狀況
- 查看當地未來一周的天氣狀況
- 點擊加號追蹤該城市的天氣狀況

<br>

# 專案初始化
## **前置作業**
已安裝 node 和 npm

<br>

## **安裝**
1. Clone 專案
```
 git clone https://github.com/TallTall22/weatherApp

 cd weatherApp
```

<br/>

2. 安裝套件
```
npm install
```

<br/>

## **啟動專案**

1. 啟動專案
```
npm run dev
```

如果成功會看到 vite 啟動的 Local 網址

<br>

2. 停止專案
```
control + c
```
<br/>
<br/>
